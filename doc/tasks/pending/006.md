## 背景

T005 跑出 Mitty-style LoRA 后，需要独立的推理脚本（不依赖训练循环里的 eval 路径）和一套和 FunControl baseline 的定量对比，才能判断新方案是否更优。

Mitty 论文支持两种推理模式：
- **H2R (zero-frame)**：只给 human 演示，模型从噪声生成整段 robot
- **HI2R (first-frame)**：给 human 演示 + 1 帧初始 robot 图，更稳定

## 目标

1. 新写 `src/pipeline/mitty_infer.py`，支持 H2R 和 HI2R 两种模式
2. 在 `training_data/pair/1s` 的 eval 集上生成对照视频
3. 计算定量指标：PSNR、SSIM、MSE（相对 ground truth robot）
4. 与 Wan 2.1 Fun-Control baseline（现有 `train_lora.py` 训出的 ckpt）对比
5. 产出 `doc/step_5_mitty_vs_funcontrol.md` 汇报对比结论

## 范围

**包含：**
- `src/pipeline/mitty_infer.py`：加载 TI2V-5B + LoRA，跑 denoising，解码 VAE，输出 mp4
- HI2R 模式：支持 `--first-frame path/to/robot_frame0.jpg`
- 批量 eval：对给定目录所有样本跑推理，落盘 + 指标计算
- 指标：PSNR、SSIM、MSE（逐帧平均）
- 对比表：Mitty vs FunControl

**不包含：**
- FVD（需要 InceptionV3 特征，工作量大，放二期）
- Human-preference 评估
- 新训 FunControl baseline（用 T003 已有的）

## 实施计划（分步骤）

### Step 1: 推理脚本 `mitty_infer.py`

- CLI：
  ```
  python -m src.pipeline.mitty_infer \
    --lora path/to/step-N.safetensors \
    --human-video path/to/human.mp4 \
    --first-frame path/to/robot0.jpg \  # 可选，HI2R 模式
    --num-frames 17 \
    --cfg 5.0 --steps 30 \
    --output out.mp4
  ```
- 模型加载：TI2V-5B DiT (FP8 可选) + VAE + LoRA merge/adapter
- Prompt：用训练时的默认 prompt
- 构造 timestep 同训练（human 段=0，robot 段=noise level t）
- Denoising：FlowMatchScheduler with "Wan" template, shift=5.0
- VAE decode → ffmpeg encode mp4

### Step 2: 批量 eval + 指标

- 新写 `src/pipeline/mitty_eval.py`（或作为 `mitty_infer.py --batch` 子模式）
- 输入：eval set 的 manifest + LoRA ckpt
- 对每条样本跑推理 → 与 GT robot 逐帧计算 PSNR/SSIM/MSE
- 输出汇总 csv：`step, sample_id, psnr, ssim, mse, mode(H2R/HI2R)`

### Step 3: 和 FunControl 对比

- 用 T003 训出的 FunControl LoRA 在相同 eval 集上跑推理
- 统一指标计算代码
- 合并到同一张表

### Step 4: 文档

- `doc/step_5_mitty_vs_funcontrol.md`：
  - 两种方案的简要原理（一张图）
  - 定量表：N 个样本的 PSNR/SSIM/MSE 均值 + 标准差
  - 定性对比：每种方案挑 3 个好 / 3 个坏样例，配图
  - 结论和 next steps

## 验收标准

1. `mitty_infer.py` 单样本推理能输出 mp4，可观看
2. HI2R 模式下 generated 视频的第一帧与 `--first-frame` 输入高度一致
3. 批量 eval 完成，csv 有完整 PSNR/SSIM/MSE
4. 对比表显示 Mitty 和 FunControl 在相同测试集上的 metric 差异
5. doc 给出明确结论（谁优、在什么条件下优、是否继续投入 Mitty 路线）

## 测试要求

- `mitty_infer.py` 跑 3 个样本（1 H2R + 2 HI2R）手动肉眼验证
- 批量 eval 跑完 eval set（~8 样本），csv 完整
- FunControl baseline 推理 + 同套指标
- 完整对比表 + 定性挑样案例落到 doc
