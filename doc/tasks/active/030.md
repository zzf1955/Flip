## 背景

当前训练循环（`src/pipeline/train.py`）在 `eval_video_steps` 触发时生成 eval 视频（gen/gt/ctrl），但只上传到 W&B，不计算量化指标。离线工具 `src/tools/eval_metrics.py` 有完整的 PSNR/SSIM/LPIPS/FID/FVD 实现，但需要训练后手动执行。

当前 eval 集仅 4 个 in_task + 4 个 OOD 样本，FID/FVD 在此样本量下统计意义很弱（2048 维 Gaussian 拟合 68 帧），因此在线指标聚焦逐对可计算的指标。

## 目标

训练过程中每次生成 eval 视频后，自动计算 PSNR / SSIM / LPIPS / CLIP Score，记录到 CSV 和 W&B，无需手动触发离线工具。

## 范围

**包含：**
- 新建 `src/core/eval_metrics.py`，封装 LPIPS + CLIP Score 模型和指标计算逻辑
- 修改 `src/pipeline/train.py`，在 eval 视频生成后调用指标计算
- 指标记录到 train.csv（新列）和 W&B（eval/ 前缀）
- CLIP Score 使用 `transformers` 库的 `openai/clip-vit-large-patch14`

**不包含：**
- 不修改 `src/tools/eval_metrics.py`（离线工具保持独立）
- 不在线计算 FID/FVD（样本量不足，等后续补数据）
- 不改变 eval 视频生成逻辑或数据集划分

## 实施计划（分步骤）

### Step 1: 新建 `src/core/eval_metrics.py`

创建 `OnlineMetrics` 类：
- 持有 LPIPS (VGG16) + CLIP (ViT-L/14) 模型，懒加载
- `compute_step(step_dir: str) -> dict` 方法：
  1. 扫描 step_dir 下的 `gen_XX.mp4` + `gt_XX.mp4` 配对
  2. 用 ffmpeg 读取帧（复用 `src/tools/eval_metrics.py` 的 `read_video_frames` 逻辑）
  3. 逐帧计算 PSNR (skimage) / SSIM (skimage) / LPIPS (VGG16) / CLIP Score (cosine similarity)
  4. 返回 `{"psnr": float, "ssim": float, "lpips": float, "clip_score": float}`

LPIPS 实现从 `src/tools/eval_metrics.py` 提取（VGG16 五层特征 L2 距离）。

CLIP Score：用 `transformers.CLIPModel` + `CLIPImageProcessor`，提取 gen 和 GT 的 image embedding，计算 cosine similarity 后跨帧和样本取均值。

### Step 2: 修改 `src/pipeline/train.py`

1. 导入 `OnlineMetrics`
2. 在 model 加载后、训练循环前创建 `OnlineMetrics(device)` 实例（仅 rank 0）
3. 在 eval 视频生成后（barrier 之后、is_main 块内），调用 `compute_step()` 计算指标
4. CSV headers 扩展 8 列：`eval_psnr_in_task`, `eval_ssim_in_task`, `eval_lpips_in_task`, `eval_clip_in_task`, 同样 4 列给 OOD
5. W&B 记录：`eval/psnr_in_task`, `eval/ssim_in_task` 等

### Step 3: 验证

在已有实验目录上测试 `OnlineMetrics.compute_step()` 输出是否与离线工具一致（PSNR/SSIM/LPIPS 对比）。

## 验收标准

1. 训练 CSV 在 eval_video 行包含 8 个新指标列，数值合理
2. W&B dashboard 可看到 eval/psnr_in_task, eval/ssim_in_task 等曲线
3. PSNR/SSIM/LPIPS 数值与离线工具 `eval_metrics.py` 一致（容差 < 0.01）
4. CLIP Score 在 [0, 1] 范围，gen vs GT 典型值 > 0.8
5. DDP 多卡训练时仅 rank 0 计算指标，不影响其他 rank

## 测试要求

- 对已有实验 `Mitty-transfer2LoRA-124d_r256_2000s_0423_1931` 的 eval/in_task/step-2000 目录运行 `OnlineMetrics.compute_step()`，对比离线工具结果
- 单 GPU 短训练（10 步）验证 CSV 和 W&B 输出
