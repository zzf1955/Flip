## 背景

人体 Mesh 辅助 Seedance 路线中，SMPLH overlay 与 G1 robot mesh 的局部贴合仍不理想。问题不只在手部，也包括整体 body scale、手臂长度、mesh 前后偏移、手部尺寸和手指动态姿态。需要用定量指标和渲染图更激进地优化全身人体 Mesh 贴合。

## 目标

1. 建立 G1 robot mesh mask 与 SMPLH human mesh mask 的 2D 投影贴合指标。
2. 搜索/优化 body scale、hand scale、全身前后/上下偏移、可选 wrist correction 等参数。
3. 将最佳参数接入人体 overlay 主线。
4. 输出优化前后对比图和指标，兼顾全身 IoU 与手部局部覆盖。

## 范围

包含：
- `segment_pipeline.py` 动态手型和优化后的全身/手部 shape 参数。
- 全身贴合 metric/debug 工具。
- 文档记录参数、验证命令和指标。

不包含：
- 重跑完整 Seedance API。
- 修改 SAM2/inpaint 算法。

## 实施记录

- 新增 `src/tools/body_fit_search.py`：
  - target：G1 FK robot mesh body mask 和 hand mask。
  - prediction：SMPLH no-head body mask 和左右手 face mask。
  - metrics：body/hand IoU、recall、precision，以及加权 score。
- 搜索 `Inspire_Put_Clothes_Into_Basket ep000 seg01 f0/30/60/90`：最佳 `body_scale=0.75 hand_scale=2.0 root_x=0.02 root_z=0.0`，考虑跨样本泛化后未采用过大的 2.0 手部。
- 搜索 `Inspire_Collect_Clothes_MainCamOnly ep000 seg02 f0/30/60/90`：最佳 `body_scale=0.75 hand_scale=1.6 root_x=0.02 root_z=0.02`。
- 折中接入主线参数：`body_scale=0.75 hand_scale=1.8 root_offset_g1=[0.02, 0.0, 0.0]`。
