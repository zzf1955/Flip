## 目标

构建 Qwen-Image-Edit 无 mask 指令式编辑的 ComfyUI workflow 脚本，与 Flux Fill 对比编辑效果。

## 范围

- 编写 `scripts/comfyui_qwen_edit.py`
- 复用 ComfyUIClient
- 无需 mask，通过文本指令编辑
- 对相同样本帧运行，与 Flux Fill 对比

## 实施计划（分步骤）

1. 参考 `archive/3-30-data_search_and_premtive_experiment/scripts/comfyui_qwen_edit.py` 编写新脚本
2. 复用 Task 002 中的 ComfyUIClient
3. 构建 Qwen Edit workflow：Load Qwen Model → Load Qwen VL Encoder → Image Input → Text Instruction → KSampler → Save
4. Prompt: `"Replace the robot with a realistic human performing the same action. Keep background unchanged."`
5. CLI 接口：`--image`, `--output-dir`, `--prompt`, `--steps`, `--denoise`
6. 输出到 `data/leverb_edited/qwen/`

## 验收标准

- 脚本可对单张帧执行 Qwen Edit 并保存结果
- 支持自定义 prompt 和参数
- 输出图像与输入同尺寸

## 测试要求

- 对 Task 001 提取的帧运行，ComfyUI 无报错
- 输出图像可正常查看
- 与 Flux Fill 结果进行对比
