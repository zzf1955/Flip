## 背景

当前 `hand_patch` 系统只处理手部区域的 latent-space 权重图。用户要将此扩展为全身机器人 patch，生成降质视频 + latent mask，用于外观学习训练。训练时 model 从降质 condition 恢复原始外观，学会记忆机器人的外观细节。

## 目标

1. 新建 `src/pipeline/robot_patch.py`，生成三类输出：原始机器人视频、降质视频、latent mask (patch.pth)
2. 支持三种降质模式：blur、noise、mean
3. 输出格式与 `mitty_cache.py` / `train.py` 完全兼容

## 范围

**包含**：
- `src/pipeline/robot_patch.py` 新脚本
- `src/core/config.py` 添加 `ROBOT_PATCH_DIR` 常量

**不包含**：
- 不修改训练代码（train.py, train_mitty.py, mitty_model_fn.py）
- 不修改 mitty_cache.py

## 实施计划（分步骤）

1. 在 `config.py` 添加 `ROBOT_PATCH_DIR` 常量
2. 创建 `robot_patch.py`，实现：
   - segment 发现与 train/eval 划分（复用 make_robot_pair.py）
   - FK 全身 mask 渲染（复用 render_mask）
   - 三种降质模式实现
   - 30fps→16fps 帧重采样
   - latent space mask 计算
   - 视频写入 + metadata/source_map 生成
3. Smoke test：单 task 验证三种 degrade 模式
4. 全量生成

## 验收标准

- `python -m src.pipeline.robot_patch --task all --degrade blur` 正常运行
- 输出 video/ (17帧) + control_video/ (17帧降质) + patch/ (.pth mask shape (5,30,40))
- metadata.csv 格式与 make_pair.py 一致
- `mitty_cache.py` 可直接编码生成的 pair

## 测试要求

- 单 task + 2 segment smoke test
- 检查输出视频帧数、patch.pth 格式
- 三种 degrade 模式各自验证
