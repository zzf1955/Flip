## 背景

identity LoRA（Stage 1）已完成训练，在 q/k/v/o 注意力层上学习了恒等映射。现在需要添加新的训练阶段，在 FFN 层加 LoRA 学习从降质视频重建完整视频，让模型记住机器人外观细节。

## 目标

1. `train_mitty.py` 支持 `--merge-lora` 参数，将 identity LoRA 合并到基础权重
2. 支持 `--lora-target-modules "ffn.0,ffn.2"` 在 FFN 层上训练新 LoRA
3. `mitty_cache.py` 支持无 metadata.csv 时自动生成

## 范围

- 包含：train_mitty.py 合并逻辑、CLI 参数、mitty_cache.py fallback
- 不包含：数据生成、实际训练运行

## 实施计划（分步骤）

1. `train_mitty.py` 新增 `merge_lora_into_weights()` 函数
2. `MittyTrainingModule.__init__` 加入 merge_lora_path/merge_lora_rank 参数
3. `train()` 传入新参数，日志输出合并信息
4. `main()` 新增 CLI 参数及路径解析
5. `mitty_cache.py` 添加 metadata.csv 自动生成 fallback

## 验收标准

- `--merge-lora` 指定 identity checkpoint 后，base weight 被修改
- `--lora-target-modules "ffn.0,ffn.2"` 仅 FFN LoRA 可训练（~100M 参数）
- 无 metadata.csv 时 mitty_cache.py 自动生成并继续

## 测试要求

- 单 GPU smoke test 5 steps 确认 loss 正常
